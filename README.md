## 🏥 What is Maple Hospital?

Maple Hospital is an engaging role-playing experience where you can work various jobs in a hospital setting. Whether you're treating patients as a pharmacist, navigating the hospital floors, or learning the ins and outs of hospital operations, every role is important.

## 📖 How to Use This Course

Each module is self-contained and can be completed at your own pace. Within each module, you'll find:

- Visual guides
- Common mistakes to avoid
- Practice tips
